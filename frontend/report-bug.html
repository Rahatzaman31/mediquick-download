Report a Bug

Help us improve by reporting issues

We appreciate you taking the time to report bugs and help us improve MediQuick. Your feedback is valuable in making our app better for everyone.

How to Report a Bug

1. Check if it's a known issue

Before reporting, check our FAQ page to see if the issue is already known and being addressed.

2. Gather information

Please include the following details:

  • Device model and Android version
  • App version number
  • Steps to reproduce the bug
  • Screenshots or screen recordings (if applicable)
  • Error messages you received
  • When the issue occurred

3. Contact our support team

Send your bug report to:

Common Issues

App Crashes

If the app crashes frequently, try clearing the app cache or reinstalling the app. Make sure you're using the latest version.

Login Problems

Ensure you're using the correct phone number and password. Use the "Forgot Password" option if needed.

Notification Issues

Check that notifications are enabled for MediQuick in your device settings.

Payment Errors

Verify your payment method and internet connection. Contact support if the issue persists.

Bug Priority Levels

Critical

App crashes, data loss, security issues - We address these immediately.

High

Features not working, significant functionality issues - Fixed in the next update.

Medium/Low

Minor UI issues, enhancement requests - Scheduled for future releases.

What Happens After You Report?

  1. We acknowledge receipt of your bug report within 24 hours
  2. Our team investigates and reproduces the issue
  3. We prioritize the bug based on severity
  4. You receive updates on the fix status
  5. The fix is released in an app update

Thank You!

Your contributions help make MediQuick better for everyone. We appreciate your patience and support.
